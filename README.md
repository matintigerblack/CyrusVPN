# 🏛️ Cyrus VPN

یک VPN هوشمند با قابلیت تعویض خودکار کانفیگ

## ✨ امکانات

- 🔄 **تعویض خودکار** - وقتی اتصال ضعیف شد، بدون دخالت کاربر کانفیگ عوض میشه
- 📡 **چند کانال تلگرام** - کانفیگ از چندین کانال جمع‌آوری میشه
- ⚡ **تست سرعت** - همه کانفیگ‌ها تست و رتبه‌بندی میشن
- 🩺 **مانیتور سلامت** - اتصال لحظه‌به‌لحظه بررسی میشه
- 🎨 **تم طلایی کوروش** - رابط کاربری زیبا

## 📱 دانلود

آخرین APK رو از بخش **Actions** دانلود کنید.

## 🛠️ ساخته شده با

- Flutter 3.22
- flutter_v2ray
- sing-box / xray-core
